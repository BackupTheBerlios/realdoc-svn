RealDoc - Simple API documentation for Python

Project is hosted on BerliOS RealDoc project site.
RealDoc can parse Python source code and generate various output, like text, LaTeX, DocBook or RealDoc-XML. RealDoc is fast and does not require packages or modules to be imported (i.e executed) which means that you can get API documentation from projects that require tedious setup without the actual setup. Just let RealDoc read it's source code.

Example output

Since this is a CLI utility and the output is text, there are no screenshots. But here are some examples of XHTML generated using the built-in XSLT-support and RealDoc-XML serialization.

realdoc -t "The RealDoc API" -c "Copyright 2006 Peter Gebauer, licensed as Public Domain" -x -o website/examples/realdoc-api src/

The result is available in examples/realdoc-api/

realdoc -t "The Python 2.4 logging-module" -x -o website/examples/python2.4-logging /usr/lib/python2.4/logging/

The result is available in examples/python2.4-logging/

Requirements:

  • Python 2.4
  • GNU make (optional; you can preform a manual install)
  • libxml2/libxslt (optional; only used with the CLI -x option)
